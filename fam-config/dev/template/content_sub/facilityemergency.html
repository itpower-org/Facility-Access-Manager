Hint

Here you can set a sudden failure of a facility.
All booking requests overlapping the expected duration will be canceled on a failure alert.
If your facility is seperated into units, you can set the failure only on the entire facility (on all units) and not just for parts of it.
You can stop the failed status before the expected time run out. But this will not annul the cancelation of affected booking requests.
If you want to set a regular maintenance on a facility you are responsible for, $util.html.a('systemfacilityavailability').setContent('please use this link') instead of this page.
